Algorithm Information
Left Wall Follower Right Wall Follower Breadth-First Search Depth-First Search A* Search

These algorithms demonstrate different approaches to pathfinding in mazes:

  • Wall Followers - Simple algorithms that follow the left or right wall until reaching the exit. Only works in simply connected mazes.
  • Breadth-First Search (BFS) - Guarantees the shortest path by exploring all possible routes level by level.
  • Depth-First Search (DFS) - Explores as far as possible along a branch before backtracking. May not find the shortest path.
  • A* Search - An informed search algorithm that uses a heuristic to guide exploration towards the goal. Efficient and optimal.
